# bin-get

Get binaries from Github in a safe and secure way!

## Usage

`bin-get install <user/repo> <version>`

Examples: 
`bin-get install hadolint/hadolint v2.10.0`
`bin-get install hadolint/hadolint`
`bin-get install helm/helm v3.9.1`
